Introduction

SAPUI5 is a state-of-the-art JavaScript UI library that:
  • ... provides RIA-style AJAX features
  • ... supports the SAP product standards
  • ... is based on Open Source libraries like jQuery
  • ... offers an extensibility concept
  • ... allows theming and customer branding
For more information visit us at https://sapui5.hana.ondemand.com/sdk
